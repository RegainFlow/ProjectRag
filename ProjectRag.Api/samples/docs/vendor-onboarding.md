# Vendor Onboarding

New vendors must complete security, privacy, and finance review before receiving purchase approval.

The requester must provide the vendor name, business purpose, expected annual spend, data access level, and contract owner.

Vendors with access to customer data require a security questionnaire and a data processing agreement.

Finance verifies tax forms, payment details, and purchase order requirements before the vendor is activated.

Vendor access must be reviewed annually and removed when the vendor relationship ends.
