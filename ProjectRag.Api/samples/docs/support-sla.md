# Support SLA

Support requests are assigned a severity level based on customer impact.

Severity 1 incidents involve a production outage, data loss, or critical security issue. Initial response target is 30 minutes.

Severity 2 incidents involve major degraded functionality with no reasonable workaround. Initial response target is four business hours.

Severity 3 incidents involve minor defects, questions, or requests with an available workaround. Initial response target is two business days.

Customers should provide logs, screenshots, affected users, reproduction steps, and business impact when opening a support case.
