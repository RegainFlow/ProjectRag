# Security Policy

All employees must use company-approved devices for access to production systems.

Multi-factor authentication is required for email, source control, cloud infrastructure, customer systems, and administrative tools.

Production secrets must be stored in an approved secret manager. Secrets must not be committed to source control, shared through chat, or stored in plain text files.

Security incidents must be reported to the security team within one hour of discovery. Examples include lost devices, suspected phishing, leaked credentials, malware alerts, or unauthorized access.

Access reviews are performed quarterly. Managers must confirm that each user still requires their assigned permissions.
