# Access Control Policy

Access is granted according to least privilege. Users should receive only the permissions required for their current role.

New access requests must include the system name, requested role, business justification, manager approval, and expiration date when temporary access is needed.

Administrative access requires multi-factor authentication and security team approval.

Access must be removed within one business day after employee termination or role change.

Privileged access is reviewed monthly. Standard application access is reviewed quarterly.
