# Password Policy

Passwords must be at least 14 characters long and must not reuse passwords from personal accounts.

Employees should use a company-approved password manager to generate and store passwords.

Password rotation is required when a credential is suspected to be compromised. Routine password expiration is not required when multi-factor authentication is enabled.

Shared passwords are not allowed. Service accounts must use managed secrets or approved credential vaults.

Failed login alerts should be reviewed by the security team when repeated attempts occur from unfamiliar locations.
